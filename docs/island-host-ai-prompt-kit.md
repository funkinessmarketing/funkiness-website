Copy any prompt into ChatGPT (chatgpt.com) or Claude (claude.ai). Fill in every [PLACEHOLDER] with your own details. The more specific you are, the better the result.

## CATEGORY 1: LISTING DESCRIPTIONS (10 prompts)

---

### Prompt 1 - Airbnb Listing Description

```
You are a top-tier hospitality copywriter specializing in Caribbean vacation rentals. Write a compelling Airbnb listing description for my property. Use vivid, sensory language that makes the reader feel the trade winds and see the turquoise water. Avoid clichés like "paradise," "gem," and "home away from home."

Property details:
- Name: [PROPERTY NAME]
- Location: [NEIGHBORHOOD/AREA on ARUBA/BONAIRE/CURACAO]
- Type: [apartment/villa/guesthouse/studio]
- Bedrooms: [NUMBER] / Bathrooms: [NUMBER]
- Max guests: [NUMBER]
- Key features: [LIST 4-6 FEATURES, e.g., private pool, ocean view, walk to beach]
- Unique selling point: [WHAT MAKES THIS PROPERTY DIFFERENT]

Structure the description as:
1. An opening hook (2-3 sentences, no questions)
2. The experience of staying here (what does a day feel like?)
3. The space (specific highlights only, not a room-by-room tour)
4. The location advantages
5. A closing line that creates urgency without being pushy

Keep it under 500 words. Write in a warm, confident tone.
```
